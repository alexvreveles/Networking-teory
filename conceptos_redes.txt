IP publica: Haciendo la analogia es como si fuera la direccion de un hogar donde un ruter es instalado el cual le dara red a los dispositivios que se encuentre.

IP Privada: Siguiendo con la anlogia la ip privada serian como las habitaciones de este lugar donde se entregaran direcciones privadas a cada dispositivo.

NAT: Es un proceso que permite que multiples IPs privadas compartan una sola direccion publica y se puedan conectar a internet,donde el el rooter es la puerta de enlace de todo esto.

DNS: Es un servidor de internet donde esta almacenanado una serie de IPs, cundo quieras acceder a una IP especifica el DNS te conecta te da la direccion para que te puedad conectar a ella.

Puerto: Es un numero determinado que se le brinda a cada aplicacion dentro de un ordenador con el fin de identidicarlo.

Socket: Es esa union de la ip del ordenador y el puerto, es lo que se denomina socket. Se representa de manera numerica, donde se acomada el numerol de ip seguido de dos puntos y al final el puerto. Ejemplo 28.156.190.20:80
